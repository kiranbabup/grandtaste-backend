# Ecommerce Backend

This is the backend API for a modern ecommerce application built with Node.js, Express, and MongoDB.

## Features
- JWT Authentication
- Google OAuth 2.0 Integration
- Role-based Access Control (Admin/User)
- Asset Management with Cloudinary
- Real-time Order Tracking

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/manojkumar8985/ecommerce-backend.git
   ```
2. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
3. Configure environment variables in `.env`.

### Running Locally
```bash
npm run dev
```
